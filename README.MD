# 🚀 AI Resume Analyzer

A full-stack web application that analyzes resumes using AI to provide insights, improvements, and ATS-based feedback. Users can either paste their resume text or upload a PDF to receive instant analysis.

---

## 🌐 Live Demo

👉 https://ai-resume-analyzer-86bbyv8zk-harshitatantia1-9637s-projects.vercel.app/

---

## 📌 Features

* Upload resume (PDF) or paste text
* AI-powered resume analysis
* ATS score evaluation
* Resume improvement suggestions
* Custom prompt-based analysis

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Axios
* CSS

### Backend

* Node.js
* Express.js
* PDF parsing (for resume upload)

### Deployment

* Frontend: Vercel
* Backend: Render

---

## ⚙️ How to Run Locally

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Harshita820/ai-resume-analyzer.git
cd ai-resume-analyzer
```

### 2️⃣ Run Backend

```bash
cd server
npm install
npm start
```

### 3️⃣ Run Frontend

```bash
cd client
cd ai-resume-analyzer
npm install
npm start
```
